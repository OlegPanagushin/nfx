OwnsAssets

Returns true when all assets (such as brushes, pens, fonts) are owned by the canvas instance and they can not be used/persisted beyond the scope of this canvas lifecycle. Some libraries allow to create objects and cache them for subsequent use, whereas others (e.g. Windows classic GDI) mandate that all graphics handles belong to the particular canvas instance and get invalidated when that instance gets releases via Dispose()
